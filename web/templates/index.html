Cloud Genius
A Genius Textual Analysis Tool in the Cloud

Cloud Genius is a web service that provides an easy to use sentiment analysis tool for anyone to use. Just paste a link to any song or article posted on genius.com and the tool will present a simple positive or negative sentiment analysis based on the words that are within the song or the article.

The sentiment analysis will be presented in a line chart and will show the polarity of the article or song, sentence by sentence.
